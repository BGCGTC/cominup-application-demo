# Comin' Up No-Login Application — Clickable Demo

Interactive mockup of the proposed no-login Comin' Up membership application
for Boys & Girls Clubs of Greater Tarrant County. Single-page HTML demo,
no backend.

All names, phone numbers, email addresses, and applications shown in the
demo are fictional sample data.

The production version is planned as a custom Lightning Web Component on
the Experience Cloud guest site.
